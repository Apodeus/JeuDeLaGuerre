Detailed Description

Object to be extended by Composite of IRules. Check the validity of all rules added by RuleComposite#add(IRule) and allows the use of rule dependencies if those are not respected. It's recommended to use instead of checking atomic rules directly on the ruleEngine.RuleChecker.
